Build your degree

This tool was developed by the School of Chemical and Physical Sciences (SCPS) to facilitate degree planning for Physics students. It provides a graphical interface for a typical BSc structure (i.e. over 3 years, 2 trimesters each year). Courses pre-requisites can be visualised and major requirements for many science major can be checked. Examples of full programmes (such as double majors) including Physics can be visualised easily.

It was not originally intended to be shared, but as it may be useful to others, it is shared here as-is.

This not an official source of information for courses. Users should always refer to the VUW website for the latest correct information on courses.

Some important things to note:

  • This tool has not been designed as a modern multiplatform/responsive layout. Its probably works only on modern browsers (Firefox and Chrome), and only on devices with sufficiently large screens. There is likely to be a few layout hiccups here and there. The focus is on functionality, not aesthetics.
  • The list of courses is based on courses offered in 2020/2021 during T1 or T2 (no T3 courses) and only include recent changes related to physics. A few 30pts courses running over both T1/T2 are shown as T1 only courses. Courses over the mid-trimester break are shown as T1 only courses. Some courses may be missing...
  • Courses from the faculty of Law are not included (there are too many courses with the same label for the current layout).
  • Major requirements are only available for a few Science majors. Many of them have not been tested carefully, so always double check with the official information on the VUW website.
